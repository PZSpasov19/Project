About Us

History of the Wild Horror Circus

The Wild Horror Circus is the oldest and darkest circus company in the world. Since 1767, it has attracted many fans of thrills and strange things. For centuries, many rumors have circulated about the Wild Horror Circus. There are several versions of this story.

According to testimonies, many students of the Wild Code School have mysteriously disappeared. Some people also say that our circus artists are not alive. Their souls wander in our world since they signed their contract with the circus. It is also said that these artists give the children who come to see them as food for their horrible lions.

But don't worry, these are just rumors!
